ШАГ 1: СБОРКА ПРОЕКТА
═══════════════════════════════════════════════════════════════════════════════

Откройте терминал/командную строку в папке SpriteAtlasForge и выполните:

Windows (PowerShell/CMD):
  dotnet restore
  dotnet build
  dotnet run --project src\SpriteAtlasForge.App

Linux/macOS:
  dotnet restore
  dotnet build
  dotnet run --project src/SpriteAtlasForge.App

═══════════════════════════════════════════════════════════════════════════════
ШАГ 2: ПОПРОБУЙТЕ ПРИМЕР
═══════════════════════════════════════════════════════════════════════════════

После запуска приложения:

1. File → Open Image (Ctrl+O)
2. Выберите: examples/war_and_purr_assets.png
3. Нажмите "Add" для создания Grid Group
4. Настройте сетку:
   - Origin X: 600, Origin Y: 20
   - Cell Width: 160, Cell Height: 160
   - Columns: 8, Rows: 8
5. Нажмите "Generate Frames"
6. File → Export Atlas (Ctrl+E)

Решение: Установите .NET 8.0 SDK с https://dotnet.microsoft.com/download

Проблема: Ошибки при restore
Решение: dotnet restore --source https://api.nuget.org/v3/index.json

Проблема: Приложение не запускается на Linux
Решение: sudo apt-get install libfontconfig1 libicu-dev

📄 ФОРМАТ ЭКСПОРТА
═══════════════════════════════════════════════════════════════════════════════

Экспортируется runtime-ready JSON atlas для игровых движков.

Пример структуры:
{
  "version": 1,
  "texture": "sprites.png",
  "groups": {
    "player": {
      "type": "character",
      "animations": {
        "idle": {
          "fps": 8,
          "loop": true,
          "frames": [...]
        }
      }
    }
  }
}
